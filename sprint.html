Team Rules

How we run this

Compass first. Everything else parallel.
Phases I and II are almost entirely Compass. Partners and AI run alongside but cannot block it.
Existing survey data is the starting point
Don't build from scratch. Import Zoho responses into new schema. Use years of data to validate the Score algorithm from Day 1.
Solution Partners trigger. Global Partners recommend.
Two different engines. Don't mix. Triggers fire on pain signals. Recommendations run on stack gap detection at onboarding.
Kewin sells from Day 3
Solution Partner outreach starts Week 1 using the prototype. Agreements signed before the portal exists.
API contracts before UI build
Canver publishes endpoint schemas before Mohammed builds against them.
David has 5 sign-offs only
Day 14, 28, 42, 49, 56. No tactical questions to David mid-phase.
